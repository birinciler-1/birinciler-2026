# Birinciler

مشروع ويب بسيط لعرض المنتجات وصفحة طلبية مع البحث بالكود والتصدير إلى Excel.

## المميزات
- صفحة رئيسية مع زر طلبية.
- البحث بالكود فقط.
- الانتقال بالـ Enter بين الحقول.
- إضافة وحذف الأسطر.
- تصدير الطلبية إلى Excel.

## الملفات
- `index.html`
- `birinciler-icon.jpeg`

## التشغيل
افتح `index.html` مباشرة في المتصفح، أو ارفعه على GitHub Pages.
